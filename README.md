# fullstack_developer_capstone

## Dealership Capstone Project
This is a full-stack application developed as part of the IBM Full Stack Developer Professional Certificate. 

### Project Overview
The application is a dealership reviews portal where users can:
* View a list of car dealerships across different states.
* View reviews left by other customers.
* Register and login to the platform.
* Submit their own reviews for specific dealerships.

### Tech Stack
* **Frontend:** React, HTML, CSS (Bootstrap)
* **Backend:** Django (Python)
* **Database:** NoSQL (MongoDB), SQLite
* **Deployment:** IBM Cloud Code Engine
